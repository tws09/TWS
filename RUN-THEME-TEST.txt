HOW TO RUN THE THEME TEST SCRIPT
==================================

STEP 1: Open PowerShell
------------------------
- Press Windows Key + X
- Select "Windows PowerShell" or "Terminal"
- OR search "PowerShell" in Start Menu

STEP 2: Navigate to the TWS directory
-------------------------------------
Copy and paste this command:

    cd C:\Users\Super\Desktop\TWS

STEP 3: Run the script
----------------------
Copy and paste this command:

    .\test-theme-api.ps1

If you get an error about execution policy, use this instead:

    powershell -ExecutionPolicy Bypass -File .\test-theme-api.ps1

STEP 4: Enter your credentials
-------------------------------
The script will ask you:
1. Enter your email: (type your healthcare tenant email)
2. Enter your password: (type your password - it will be hidden)

STEP 5: Watch the output
------------------------
The script will:
- Login with your credentials
- Get the current theme
- Update the theme to "blue"
- Verify the theme was saved

STEP 6: Check backend terminal
------------------------------
While the script runs, watch your backend terminal for:
- Logs starting with 🎨 (theme operations)
- Logs starting with 🔧 (theme updates)
- Logs starting with ❌ (errors)

WHAT TO LOOK FOR:
-----------------
✅ Success: Script shows "Theme persisted correctly!"
❌ Problem: Check backend logs for errors
❌ Problem: Check if tenantId is correct in logs
