ERROR HANDLING
================================================================================

[ERROR-1] Cannot Approve - Previous Step Not Approved
    Cause: Trying to approve Step 2 before Step 1
    Solution: Approve steps in sequential order

[ERROR-2] Access Denied - Not Project Member
    Cause: User not added to project
    Solution: Project manager must add user to project

[ERROR-3] Deliverable Not Ready for Approval
    Cause: Status not 'ready_approval'
    Solution: PM must mark deliverable as ready

[ERROR-4] Cannot Link Task - Task Not in Project
    Cause: Task belongs to different project
    Solution: Create task in same project first

[ERROR-5] Invalid Status Transition
    Cause: Trying to move from 'created' to 'approved' directly
    Solution: Follow status flow: created вҶ’ in_dev вҶ’ ready_approval вҶ’ approved
