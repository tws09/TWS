# TWS Website - The Wolf Stack

A beautiful, modern, and responsive website showcasing TWS (The Wolf Stack) - a comprehensive multi-tenant ERP platform.

## 📁 Files Created

- `index.html` - Main website HTML file
- `styles.css` - Complete styling with modern design
- `script.js` - Interactive JavaScript functionality
- `README_WEBSITE.md` - This file

## 🚀 How to Use

### Option 1: Open Directly in Browser
Simply double-click `index.html` to open it in your default web browser.

### Option 2: Use a Local Server (Recommended)

#### Using Python:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```
Then open: `http://localhost:8000`

#### Using Node.js (http-server):
```bash
npx http-server -p 8000
```
Then open: `http://localhost:8000`

#### Using PHP:
```bash
php -S localhost:8000
```
Then open: `http://localhost:8000`

Edit the CSS variables in `styles.css`:
```css

    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... */
}
```

- Company information

- Feature descriptions
- Statistics

### Add Sections
Simply add new sections following the existing pattern in `index.html`.

## 🌐 Deployment

### GitHub Pages
1. Push files to a GitHub repository
2. Go to Settings > Pages
3. Select main branch
4. Your site will be live at `username.github.io/repository-name`

### Netlify
1. Drag and drop the folder to Netlify
2. Or connect your GitHub repository
3. Automatic deployment

### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project directory
3. Follow the prompts
